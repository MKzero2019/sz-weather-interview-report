# 深圳天气数据分析项目

**项目名称：** sz-weather-interview-report  
**分析周期：** 2026年6月20日 - 2026年7月3日（共14天）  
**分析地点：** 深圳  

---

## 项目介绍

本项目以深圳天气数据为分析对象，通过完整的数据工程流程，展示数据分析岗位所需的四大核心能力：

1. **资料搜寻** - 调研并选择合适的数据源，扩展数据指标
2. **程式码撰写** - 模块化、工程化的代码结构，面向对象编程
3. **数据处理** - 缺失值处理、异常值检测、特征工程等完整预处理流程
4. **图表分析** - 多维度可视化与统计分析，CPK过程能力分析

---

## 技术总结

### 1. 资料搜寻

- **数据源调研**：对比了Open-Meteo、和风天气、天气网等多个数据源
- **最优选择**：选用Open-Meteo API（免费开源、数据权威、指标丰富）
- **数据扩展**：从基础的4个指标扩展到8个气象指标+多个衍生特征
- **API整合**：同时调用天气API和空气质量API，合并为完整数据集

### 2. 程式码撰写

- **模块化设计**：爬虫、数据处理、可视化三大模块独立，职责清晰
- **配置分离**：请求配置与业务逻辑分离，便于维护
- **面向对象**：使用Class封装，代码结构清晰，可复用性强
- **错误处理**：包含异常捕获、超时设置、数据校验
- **代码规范**：详细的函数注释、类型提示、日志输出

### 3. 数据处理

- **完整流程**：缺失值→异常值→单位统一→格式转换→特征工程→聚合统计
- **多种方法**：IQR/Z-score异常值检测，插值/截断/删除多种处理策略
- **特征工程**：基于业务理解衍生体感温差、温湿度指数等有意义特征
- **数据质量**：每一步处理都有日志记录，可追溯、可复现

### 4. 图表分析

- **图表丰富**：箱线图、直方图、散点图、折线图、热力图、CPK分析图
- **多维度分析**：时间维度、分布维度、相关性维度、过程能力维度
- **专业分析**：准确计算CP/CPU/CPL/CPK，结合业务场景深度解读
- **可视化美学**：配色协调、标注清晰、信息层次分明

## 运行步骤

### 环境准备

```bash
# 安装依赖
pip install -r requirements.txt
```

### 完整运行流程

按以下顺序运行各模块：

```bash
# 第一步：运行爬虫获取原始数据
cd crawler
python weather_spider.py

# 第二步：运行数据清洗和预处理
cd ../preprocess
python data_clean.py

# 第三步：运行可视化（三个脚本，顺序不限）
cd ../visualize
python plot_box.py
python plot_line_scatter.py
python cpk_analysis.py
```

### 运行说明

- 所有脚本使用相对路径，必须在各自目录下运行
- 各模块独立，可单独运行某一步骤
- 数据文件和图表会自动生成到对应目录

## 数据指标说明

### 原始数据指标

| 指标 | 单位 | 说明 |
|------|------|------|
| 气温 | °C | 2米高度气温 |
| 相对湿度 | % | 2米高度相对湿度 |
| 体感温度 | °C | 考虑湿度、风速等因素的体感温度 |
| 降水量 | mm | 小时累计降水量 |
| 风速 | km/h | 10米高度风速 |
| 气压 | hPa | 地表气压 |
| PM2.5 | μg/m³ | 细颗粒物浓度 |
| PM10 | μg/m³ | 可吸入颗粒物浓度 |

### 衍生特征

| 特征 | 说明 |
|------|------|
| 体感温差 | 体感温度 - 气温，量化闷热程度 |
| 温湿度指数 | THI指数，综合评估热舒适度 |
| 昼夜 | 白天/夜晚标记，用于分组分析 |
| 是否降雨 | 降雨天/晴天标记 |
| 空气质量等级 | 优/良/轻度污染等分级 |

## CPK分析说明

### CPK简介

CPK（过程能力指数）是衡量过程输出满足规格要求能力的统计指标，广泛应用于质量管理领域。

### 计算公式

- **CP** = (USL - LSL) / (6σ) - 过程潜在能力
- **CPU** = (USL - μ) / (3σ) - 上限过程能力
- **CPL** = (μ - LSL) / (3σ) - 下限过程能力
- **CPK** = min(CPU, CPL) - 实际过程能力

### 评价标准

| CPK值 | 等级 | 说明 |
|-------|------|------|
| ≥ 1.67 | A++ | 过程能力过剩 |
| 1.33 ~ 1.67 | A+ | 过程能力充足（合格线） |
| 1.0 ~ 1.33 | A | 过程能力一般 |
| < 1.0 | B | 过程能力不足 |

### 本项目规格设定

以人体舒适范围作为参考规格：

| 指标 | 规格下限 (LSL) | 规格上限 (USL) |
|------|---------------|---------------|
| 温度 | 20°C | 35°C |
| 湿度 | 40% | 90% |
